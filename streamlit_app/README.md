# Streamlit Dashboard

This folder contains the interactive dashboard for the **AI Court Battle: ChatGPT vs Grok** project.

## Run locally

From the project root:

```bash
pip install -r streamlit_app/requirements.txt
streamlit run streamlit_app/app.py
```

## What the dashboard shows

- Full 65-question dataset
- 5 live video questions
- ChatGPT and Grok answers
- Ground-truth answer key
- Score summaries
- Final charts
- Project findings

## Deployment

You can deploy this on Streamlit Community Cloud by selecting:

```text
streamlit_app/app.py
```

as the app entry point.
